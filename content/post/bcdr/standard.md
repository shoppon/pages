# 国家标准

## RTO/RPO与灾难恢复能力等级关系

| 灾难恢复能力等级 | RTO         | RPO         |
| ----------------- | ----------- | ----------- |
| 1                 | 2天以上     | 1天至7天    |
| 2                 | 24小时以上  | 1天至7天    |
| 3                 | 12小时以上  | 数小时至1天 |
| 4                 | 数小时至2天 | 数小时至1天 |
| 5                 | 数分钟至2天 | 0至30分钟   |
| 6                 | 数分钟      | 0           |
