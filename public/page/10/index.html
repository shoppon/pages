str

**判断字符串是否为空(zero)：**[[ -z "" ]] && echo "yes" # yes

**判断字符串是否非空：**[[ -n "foo" ]] && echo "yes" # yes

**判断字符串是否相等：**[[ "foo" == "bar" ]] || echo "no" # no

**判断字符串是否相等：**[[ "foo" = "bar" ]] || echo "no" # no

**判断字符串是否不相等：**[[ "foo" != "bar" ]] && echo "yes" # yes
